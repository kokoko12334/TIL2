- vsc 키설정: json keyboardshorts에서 다음 내용 복사, 붙혀넣기

```
// Place your key bindings in this file to override the defaultsauto[]
[
  {
    "key": "ctrl+;",
    "command": "terminal.focus",
    "when": "editorFocus"
  },
  {
    "key": "ctrl+;",
    "command": "workbench.action.focusActiveEditorGroup",
    "when": "terminalFocus"
  }
]
```
